Website Site Plan

WDD 131 – Project Planning Document

Site Name

Ward Connect Newsletter

This site name was chosen because it represents a central place where ward members can stay connected and informed. The word “Connect” emphasizes communication, unity, and community while remaining neutral and appropriate for an unofficial informational resource.

Optional domain: wardconnect.org

Site Purpose

The purpose of this website is to serve as an unofficial informational digital newsletter for a local ward of The Church of Jesus Christ of Latter-day Saints. The site will provide ward announcements, upcoming events, schedules, inspirational messages, and general information for ward members. The goal is to make ward information easily accessible in a clear and organized format.

Scenarios

  • What activities or events are happening in the ward this week?
  • Where can I find the current meeting schedules and important announcements?
  • How can I view inspirational messages or reminders shared by ward leaders?

Color Schema

The following colors will be used throughout the site:

  • Navy Blue (#1F3A5F): Used for headings, navigation, and primary accents.
  • Light Gray (#F2F4F7): Used for page backgrounds and content sections.

Typography

  • Montserrat: Used for headings and navigation to provide a clean, modern appearance.
  • Arial: Used for body text to ensure readability and accessibility across devices.

CSS and JavaScript

The site will use external CSS for layout, color schema, and typography. JavaScript will be used to add interactivity such as dynamic event displays, content filtering, form submissions, and saving user preferences using localStorage. The design will be responsive and accessible on mobile and desktop devices.

Testing

This page will be evaluated using HTML and CSS validation tools, accessibility checkers, color contrast testing, and SEO best practices. Performance and responsiveness will be tested across different screen sizes and devices.
